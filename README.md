# Nike Puffer Collection 🧥

A premium, interactive web experience showcasing the latest Nike Puffer Jacket collection. Designed with a focus on immersive visuals, smooth transitions, and a seamless checkout experience.

## ✨ Features

- **Immersive Hero Section**: Interactive 3D-like slider featuring the core collection with parallax floating effects.
- **Dynamic Color Themes**: Custom-tailored gradients and backgrounds that shift to match each product's identity.
- **Detailed Checkout Flow**: A complete, aesthetic purchase journey including size selection and secure shipping details.
- **Uncompromising Mobile UX**: Fully responsive design with optimized layouts, typography, and touch-friendly interactions.
- **Motion Design**: Fluid animations powered by Framer Motion for a tactile and high-end feel.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Slider/Carousel**: [Swiper.js](https://swiperjs.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Type**: [Inter](https://fonts.google.com/specimen/Inter) & [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk)

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/nike-puffer-collection.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🎨 Design Philosophy

This project was built with a "Mood-First" approach. Every visual element—from the tracking of the headings to the timing of the product entrance animations—is calibrated to evoke a sense of premium quality and modern athletic style.

- **Clean Aesthetic**: No unnecessary clutter. Focus is purely on the product.
- **Intentional Variation**: Dynamic spacing and responsive scaling for a balanced feel on all devices.
- **User-Centric**: Intuitive navigation and clear calls-to-action.
